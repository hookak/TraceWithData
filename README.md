

1. Usage
	./ent [Entropy] [Entropy distribution] [duplication rate] [duplication distribution]

	[Entropy]
		Entropy value is between 0 and 1

	[Entropy distribution]
		Support two distribution
		0 : Uniform
		1 : Normal
	
	[duplication rate]
		rate is between 0 and 1

	[duplication distribution]
		support two distribution
		0 : Uniform
		1 : Normal
